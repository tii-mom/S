# 上岸（SHORE）

> 负债不是身份，上岸才是目标。

“上岸”是一款面向负债用户的极简行动应用：用户上传负债或直接输入金额，AI 计算上岸目标；用户每天完成一个真实任务，获得积分及 SHORE 代币权益，并通过分享上岸进度形成传播闭环。

## 核心用户路径

1. 上传负债或直接输入总额
2. AI 计算并确认上岸目标
3. 完成每日任务
4. 获得积分和 SHORE
5. 分享上岸进度

## 产品原则

- 用户 3 秒理解产品
- 首次使用不要求钱包
- 每个页面只有一个主要操作
- 首页只推荐一个今日任务
- 提交证明最多两步
- 不增加小队、等级、NFT、DAO、多链等非核心功能
