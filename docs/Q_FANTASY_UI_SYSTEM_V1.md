# SHORE 单页十八轮 Q 版关卡视觉系统 V2

## 1. 页面模型

前端仅保留一个纵向单页面。页面由一条连续弯曲路线和十八个圆形关卡节点组成，对应十八轮解锁。

页面不提供：

- 顶部导航；
- 底部导航；
- 其他功能页面；
- 可见文字内容；
- 人物、角色或宠物。

## 2. 参考转译

视觉参考上传图片中的通用结构语言：

- 深蓝紫色背景；
- 自上而下的成长路线；
- 浅色宽轨道；
- 交替分布的关卡节点；
- 漂浮的小型装饰组件；
- 终点奖励物。

不得复制参考图中的人物、品牌、Logo、文字、图标或具体美术资源。

## 3. 关卡状态

十八个节点按照三种状态展示：

- 已完成：彩色节点和绿色勾选；
- 当前轮：高亮、呼吸动画和光环；
- 未解锁：低饱和节点和锁定图标。

页面不使用可见数字标识轮次。轮次信息只写入无障碍标签。

## 6. 响应式规则

- 页面最小宽度为 320px；
- 画布最大宽度为 720px；
- 画布在手机上保持足够纵向间距；
- 关卡节点使用响应式尺寸；
- 所有节点必须位于视口宽度以内；
- 页面允许纵向滚动，不允许横向滚动。

## 8. 自动验收

E2E 必须验证：

- 页面只有一个主画布；
- 恰好十八个关卡按钮；
- 没有 header 和 nav；
- 主画布没有可见文字；
- 三个完成节点、一个当前节点、十四个锁定节点；
- 所有关卡节点不超出视口；
- 页面无横向溢出；
- 页面可纵向滚动；
- 无浏览器控制台错误。
