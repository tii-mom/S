# SHORE Q版中国奇幻视觉系统 V1

## 1. 定位

产品视觉主题为“上岸仙岛”：把负债目标理解为一段每天前进一点的轻量旅程，用原创 Q 版中国奇幻语言降低压力与学习成本。

视觉只借鉴中国网络游戏中常见的高层设计语言，不复制任何现有游戏的角色、Logo、图标、具体界面、地图、文案或美术资产。

## 2. 核心视觉词

```text
云海
仙岛
玉石
卷轴
桃花
金色任务奖励
圆润 Q 版角色
```

## 3. 色彩

- 天空青：页面背景和环境层次；
- 玉石绿：主要信息、进度和安全状态；
- 桃花粉：强调行动、品牌情绪和分享；
- 暖金色：唯一主要操作和积分奖励；
- 宣纸白：任务卡和内容容器；
- 墨绿色：正文与标题。

不得使用大面积高饱和红色制造焦虑，也不使用赌场式霓虹、老虎机或宝箱开箱视觉。

## 4. 组件语言

- 页面容器采用卷轴或玉牌式圆角卡片；
- 每页只有一个暖金色主按钮；
- 状态通过小印章、玉牌和圆形徽记表示；
- 导航固定为首页、任务、金库、我的；
- 任务详情保持三步；
- 所有数字均避免交易终端式密集展示；
- 动效只用于角色呼吸、页面进入和轻量反馈。

## 5. 原创吉祥物

吉祥物名称：岸宝。

设定：从仙岛浅滩诞生的玉色小精灵，头顶嫩叶，腰间佩戴金色救生环。它负责提醒用户今天只做一个任务。

岸宝为项目原创 SVG 角色，不使用现有游戏人物的脸型、服装、武器、配色组合或标志性姿势。

## 6. 当前页面

```text
/             首页与今日主线
/debt         手动设置负债总额
/tasks        一个主线与三个支线
/tasks/:id    三步任务与证明提交
/vault        Mock SHORE 金库
/profile      本地体验数据和未来入口
/share        隐藏具体负债金额的分享卡
```

## 7. 交互原则

- 首次进入三秒内看到“每天做一件事，离上岸近一步”；
- 未设置目标时，首页唯一主操作是“开始上岸”；
- 设置目标后，首页唯一优先任务是今日主线；
- 任务完成后立即显示积分到账；
- 金库明确标注 Mock，不诱导真实购买；
- 分享失败或用户取消时，不增加分享次数；
- 所有 Mock 状态保存在浏览器本地，后续由真实会话与 D1 替换。

## 8. 知识产权边界

禁止：

- 使用《梦幻西游》名称、Logo、字体文件或角色；
- 临摹其人物、宠物、场景、按钮、边框和图标；
- 复刻其具体首页或战斗界面结构；
- 使用搜索到的截图作为产品资产。

允许：

- 使用中国奇幻、Q 版、云海、卷轴、玉石和桃花等通用文化元素；
- 以项目原创组件、SVG 和 CSS 重新设计；
- 保持“上岸”独立的品牌、角色和产品信息架构。
